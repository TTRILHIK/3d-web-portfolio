# 3D Model Gallery

Интерактивная веб-галерея для просмотра и скачивания 3D-моделей в формате GLB.

## 📋 Описание

Проект представляет собой двухстраничный сайт для демонстрации 3D-моделей с возможностью:
- Просмотра списка моделей с превью
- Интерактивного 3D-просмотра с вращением и масштабированием
- Скачивания моделей в формате GLB

## ✨ Возможности

-  Современный адаптивный дизайн
- 🔄 Интерактивное управление камерой (вращение, масштабирование)
- 📱 Полная поддержка мобильных устройств
- ⚡ Быстрая загрузка моделей
- 🎯 Автоматическое центрирование и масштабирование моделей
- 💾 Прямое скачивание файлов

## 🚀 Установка и запуск

### 1. Клонирование/скачивание проекта

Скачайте все файлы проекта в одну папку.

### 2. Добавьте ваши файлы

**Обязательно добавьте:**
- 3D-модели в формате `.glb` в папку `models/`
- Изображения-превью в формате `.jpg` или `.png` в папку `images/`

### 3. Запуск локального сервера

**Важно:** Для корректной работы Three.js необходим локальный веб-сервер.

#### Вариант A: Python 3
```bash
# В папке проекта выполните:
python -m http.server 8000
```

#### Вариант B: Node.js (http-server)
```bash
# Установите http-server (если не установлен):
npm install -g http-server

# Запустите сервер:
http-server
```

#### Вариант C: PHP
```bash
php -S localhost:8000
```

#### Вариант D: VS Code Live Server
- Установите расширение "Live Server"
- Откройте `index.html`
- Нажмите "Go Live"

### 4. Откройте сайт

Перейдите в браузере по адресу: `http://localhost:8000`

## 📖 Использование

### Главная страница
- Просмотр сетки из 3 карточек моделей
- Каждая карточка содержит:
  - Превью-изображение
  - Название модели
  - Кнопку "Посмотреть" (переход к 3D-просмотру)
  - Кнопку "Скачать" (прямая загрузка файла)

### Страница просмотра модели
- **Вращение:** зажмите левую кнопку мыши и двигайте
- **Масштабирование:** колесико мыши или жест двумя пальцами
- **Перемещение:** зажмите правую кнопку мыши
- **Автовращение:** модель медленно вращается автоматически

## ⚙️ Настройка

### Добавление новых моделей

1. Добавьте файл `.glb` в папку `models/`
2. Добавьте превью-изображение в папку `images/`
3. Отредактируйте `index.html`, добавив новую карточку:

```html
<div class="model-card">
    <img src="images/preview4.jpg" alt="Модель 4" class="preview-image">
    <h2>Модель 4</h2>
    <div class="button-group">
        <a href="model.html?id=4" class="btn btn-view">Посмотреть</a>
        <a href="models/model4.glb" download class="btn btn-download">Скачать</a>
    </div>
</div>
```

### Изменение цветов

Отредактируйте `styles.css`:
```css
/* Основной цвет */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* Цвет кнопок */
.btn-view { background: #667eea; }
.btn-download { background: #48bb78; }
```

### Настройка 3D-просмотра

В `script.js` можно изменить:
```javascript
// Скорость автовращения
controls.autoRotateSpeed = 2.0;

// Позиция камеры
camera.position.set(0, 2, 5);

// Цвет фона сцены
scene.background = new THREE.Color(0xf8f9fa);
```

## 📝 Требования к моделям

- Формат: `.glb` (бинарный GLTF)
- Рекомендуемый размер: до 10 MB
- Поддерживаются материалы: Standard, PBR
- Текстуры должны быть встроены в файл или находиться в той же папке

## 🐛 Решение проблем

### Модель не загружается
- Проверьте консоль браузера (F12) на наличие ошибок
- Убедитесь, что файл `.glb` находится в папке `models/`
- Проверьте правильность имени файла
- Убедитесь, что запущен локальный сервер (не просто открыт файл)

### Ошибки CORS
- Обязательно используйте локальный сервер, не открывайте файл напрямую
- Проверьте, что сервер запущен в корневой папке проекта

### Модель слишком большая/маленькая
- В `script.js` автоматически вычисляется масштаб
- Можно вручную настроить в функции загрузки модели

### Медленная загрузка
- Оптимизируйте 3D-модели (уменьшите количество полигонов)
- Сожмите текстуры
- Используйте Draco компрессию для GLB
